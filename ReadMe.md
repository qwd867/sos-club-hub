这是一个基于C++，Vue3的前后端分离项目。
C++使用第三方库httplib作为网络通信手段。
Vue3使用Axios与C++服务器端进行通信。

关键：
请一定，必须遵守通用C++设计原则，适当使用设计模式，代码拥有良好的扩展性，分文件进行面向对象开发

项目需求：
1. 整个系统拥有一个登录模块，登录模块需求如下：
    a.所有用户数据先存储在SQLite数据库中，密码用SHA256做加密，请自行定义数据库需要的字段
    b.用户点击注册按钮，通过邮箱注册，注册流程可以先做简单的前后端通信即可，留一些后续扩展，暂时不用真的给邮箱发短信等待验证
    c.用户注册后，自动登录
    d.用户手动执行登录时，请将密码与数据库中的内容作比较后反馈用户登录成功或失败
    e.提供忘记密码功能
2. 登录之后先展示一个Demo页面留作以后扩展，提供登出功能回到登录页面
