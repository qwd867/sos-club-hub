# 扩展功能四：SOS团弹幕留言墙

## 功能背景

参考Bilibili的弹幕文化、Niconico的评论流、《弹丸论破》的言弹（言葉弾）概念，SOS团作为2000年代最具代表性的二次元IP之一，当然要有自己的弹幕墙。团员们可以发送吐槽、应援、经典台词复诵，甚至模仿原作中阿虚的「超长内心独白」——毕竟弹幕就是要刷屏才有趣。

> 「为什么要发弹幕？因为如果不把这句话留在屏幕上，它就会像没有存在过的普通人类一样被遗忘。」——某匿名团员

## 核心概念

### 弹幕流（Danmaku Stream）
- 页面顶部设置「开启弹幕」开关
- 开启后，彩色文字从右向左飞过整个页面（经典B站模式）
- 弹幕密度可调：稀疏/正常/密集/无尽八月模式
- 弹幕轨道：分为上/中/下三条轨道，避免重叠

### 留言板（Message Board）
- 独立的「留言墙」区块，采用Pinterest风格的瀑布流布局
- 每条留言是一张「便签纸」，带有颜色、角色头像角标、发布时间
- 支持点赞（团费赞助）、回复（吐槽接力）、置顶（春日特权）

### 弹幕内容分类
| 类型 | 颜色 | 示例 |
|------|------|------|
| 吐槽 | 白色 | 「团长又在折腾人了」「阿虚你倒是说句话啊」 |
| 应援 | 粉色/金色 | 「春日最强！」「有希我爱你！」 |
| 经典台词 | 角色代表色 | 「我对普通的人类没有兴趣」「啊哇哇...」 |
| 玩梗 | 彩虹色 | 「这不比博人传热血？」「团长你在干什么啊团长」 |
| 超长独白 | 灰色小字 | 阿虚风格的500字内心吐槽（字号自动缩小） |

## 页面设计

### 弹幕层
- `position: fixed; inset: 0; pointer-events: none; z-index: 50`
- 弹幕文字使用 `white-space: nowrap`，从 `translateX(100vw)` 移动到 `translateX(-100%)`
- 不同轨道不同速度：上轨道快（12s）、中轨道中（15s）、下轨道慢（18s）
- 弹幕密度控制同时存在的弹幕数量：稀疏10条、正常25条、密集50条、无尽八月100条

### 发送弹幕框
- 位于页面右下角（悬浮按钮，点击展开）
- 展开后是一个半透明的输入框，模仿B站播放器的发送栏
- 输入框左侧有「弹幕颜色」选择器（8色 + 彩虹色VIP）
- 右侧有「发送模式」切换：
  - 普通弹幕（飞过屏幕）
  - 高级弹幕（停留3秒后消失）
  - 顶部固定（春日团长特权，红色大字置顶5秒）

### 留言墙区块
- 位于成员展示区下方（或独立页面 `/sos/wall`）
- 瀑布流布局，每列宽度250px，自动适应列数
- 便签纸样式：
  - 轻微旋转（-2deg ~ 2deg随机）
  - 带胶带/图钉装饰
  - 不同颜色代表不同角色主题色
  - Hover时放大1.05倍，阴影增强

### 留言卡片内容
```
┌─────────────────────┐
│ [鹤屋头像] 鹤屋学姐   │ ← 用户名+角色角标
│                     │
│ 哇哈哈！这个网站太   │ ← 留言内容
│ 有趣了！鹤屋家决定   │
│ 赞助十个亿！         │
│                     │
│ ❤️ 15532  💬 8      │ ← 点赞数（无尽八月梗）+回复数
│ 2026-05-31 14:32    │ ← 时间
└─────────────────────┘
```

## 特殊弹幕事件

### 春日团长的强制弹幕
- 每隔5分钟，系统自动发送一条「春日弹幕」：
  - 「SOS团全员集合！现在立刻马上！」
  - 「实玖瑠，去换上兔女郎装！」
  - 「有希，用你的资讯操作把这道菜变好吃！」
  - 弹幕为金色，字号1.5倍，速度较慢（确保所有人都能看到）

### 全员应援模式
- 当某条留言获得超过100个赞时，触发「全员应援弹幕事件」
- 屏幕瞬间被该角色的代表色弹幕填满（持续3秒）
- 例如：长门有希的留言被爆赞 → 屏幕被紫色弹幕「有希！有希！」淹没

### 无尽八月弹幕风暴
- 开启「无尽八月模式」后，弹幕密度变为100条同时存在
- 所有弹幕内容变成重复的「去写作业」「空调温度调低点」「想吃冰西瓜」
- 持续30秒后自动恢复正常，显示「第15532次循环...结束」

### 阿虚独白模式
- 输入超过100字自动触发「阿虚独白」标签
- 弹幕字号缩小到12px，速度极快（8s飞过），颜色为灰色
- 其他用户可以点击「已阅」按钮表示读完（实际上没人会读完）

## 留言墙互动机制

### 点赞 = 「团费赞助」
- 点击爱心按钮，爱心变为角色代表色
- 点赞动画：爱心向上飘起，消散成粒子
- 自己的留言被点赞时，导航栏贡献值+1（微不足道但有趣）

### 回复 = 「吐槽接力」
- 点击回复，展开输入框
- 回复样式为嵌套便签，贴在原便签下方，形成「千层饼」效果
- 最多嵌套5层，第6层显示「吐槽能量溢出，请另开新帖」

### 置顶 = 「春日特权」
- 只有系统/管理员（春日）可以置顶
- 置顶留言带有金色皇冠标记
- 置顶位置在留言墙最顶部，不会随滚动消失（sticky）
- 置顶内容示例：「本周社团活动：寻找不可思议现象！——春日」

### 热门标签
- 留言可以带标签，如 `#团长又任性了` `#长门今天也很可爱` `#实玖瑠的cosplay`
- 点击标签筛选相关留言
- 标签云展示在留言墙顶部，字号代表热度

## 视觉风格

- 弹幕字体：优先使用「思源黑体」或系统默认无衬线字体，确保中文显示清晰
- 弹幕描边：所有弹幕带1px深色描边，确保在任何背景上都可见
- 弹幕阴影：轻微发光效果（`text-shadow: 0 0 4px currentColor`）
- 留言便签：圆角矩形，纸质纹理背景，轻微阴影
- 整体色调：保持明亮风格，与SOS团页面一致

## 技术实现建议

### 弹幕性能优化
- 使用 CSS `transform: translateX()` 而非 `left` 属性（GPU加速）
- 弹幕使用 `will-change: transform` 提示浏览器优化
- 弹幕离开视口后立即从DOM中移除（避免内存泄漏）
- 弹幕池复用：预先创建固定数量的弹幕DOM元素，循环使用

### 数据结构
```typescript
interface Danmaku {
  id: string;
  text: string;
  color: string;
  size: 'small' | 'normal' | 'large';
  position: 'top' | 'scroll' | 'bottom';
  sender: string;
  senderAvatar?: string;
  createdAt: string;
}

interface Message {
  id: string;
  author: string;
  authorCharacter: string;  // 用户选择的代表角色
  content: string;
  color: string;
  likes: number;
  replies: Message[];
  tags: string[];
  isPinned: boolean;
  createdAt: string;
}
```

### 存储方案
- 弹幕（临时）：不持久化，仅当前会话可见
- 留言（持久）：`localStorage` 存储，键名 `sos_messages`
- 预设一批经典弹幕，页面加载时自动播放，营造热闹氛围
