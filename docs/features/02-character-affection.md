# 扩展功能二：角色好感度系统（Galgame模式）

## 功能背景

参考《白色相簿2》《CLANNAD》《命运石之门》等Galgame/视觉小说的好感度系统，SOS团每位成员都有独立的好感度计量槽。当好感度达到特定阈值时，解锁角色的隐藏台词、特殊互动、甚至「个人线」结局——当然，想要攻略春日团长？做好世界被改写的准备吧。

> 「为什么我只能提升阿虚的好感度？因为原作就是这样的啊。」——系统提示

## 核心概念

### 好感度等级（以长门有希为例）
| 等级 | 称号 | 好感度范围 | 解锁内容 |
|------|------|-----------|----------|
| Lv.0 | 陌生人 | 0-9 | 基础档案可见 |
| Lv.1 | 同班同学 | 10-49 | 解锁个人简介第二页 |
| Lv.2 | 文艺社部员 | 50-99 | 解锁「有希的读书推荐」列表 |
| Lv.3 | 资讯统合思念体的关注对象 | 100-199 | 解锁隐藏台词语音（文字版） |
| Lv.4 | 特殊存在 | 200-349 | 解锁《消失》IF线剧情片段 |
| Lv.5 | 唯一的理解者 | 350-499 | 解锁有希的私服立绘（占位图） |
| MAX | 资讯统合思念体的意志 | 500+ | 解锁最终告白事件，有希会说出那句「欢迎回来」 |

### 互动方式
每位角色的互动方式不同，贴合原作性格：

**凉宫春日**
- 点击方式：连续快速点击（春日喜欢热闹）
- 互动文案：「团长发现了你！」「SOS团需要你的才能！」
- 特殊机制：如果30秒不点击，春日会自动降低好感度（团长不耐烦了）
- MAX事件：春日说「好吧，特别允许你坐我旁边」

**长门有希**
- 点击方式：长按（有希需要时间处理资讯）
- 互动文案：「...」「收到」「资讯更新中」
- 特殊机制：连续7天每天互动1次，触发「《消失》纪念事件」
- MAX事件：有希摘下眼镜说「我有话要告诉你」

**朝比奈实玖瑠**
- 点击方式：轻轻点击（实玖瑠容易受惊）
- 互动文案：「啊哇哇...」「这是禁止事项...大概」「茶泡好了」
- 特殊机制：好感度提升时随机触发cosplay换装（女仆/护士/兔女郎图标变化）
- MAX事件：实玖瑠哭着说「能遇见你...真的是太好了」

**古泉一树**
- 点击方式：滑动（古泉喜欢推理的动感）
- 互动文案：「有趣」「这是机关的指示...开玩笑的」「和团长相处得还好吗？」
- 特殊机制：每次互动后古泉会反问一个哲学问题，答对双倍好感
- MAX事件：古泉第一次收起微笑说「你...真的很有趣」

**阿虚**
- 点击方式：吐槽式点击（点击后显示吐槽对话框）
- 互动文案：「喂，别点我了去点团长啊」「长门，这家伙有点奇怪」「...（内心独白300字）」
- 特殊机制：阿虚的好感度提升最慢，但永远不会降低（常识人的包容）
- MAX事件：阿虚说「...算了，就这样吧。谁让我是叙述者呢」

## 好感度增长机制

### 基础互动
- 每次点击/长按/滑动：+1～+3 好感度（随机浮动）
- 每日首次互动：双倍好感度
- 连续7天互动同一角色：第7天触发「羁绊事件」，+50好感度

### 特殊事件（彩蛋）
| 触发条件 | 事件名 | 效果 |
|----------|--------|------|
| 12月18日提升有希好感 | 《消失》之日 | 有希好感+100，显示特殊对话 |
| 夏季提升春日好感 | 无尽夏日常备 | 春日好感+50，但其他角色好感-10 |
| 同时提升实玖瑠和鹤屋好感 | 下午茶会 | 双角色同时+30 |
| 阿虚好感达到MAX后提升春日 | 最后的吐槽 | 春日好感+20，阿虚好感不变（已MAX） |
| 所有角色好感同时达到Lv.3 | SOS团全员集合 | 解锁团建合照（占位图） |

### 好感度降低
- 春日：30秒无互动 -5，点击其他角色 -3（嫉妒）
- 有希：永远不会降低（外星人的耐心是无限的）
- 实玖瑠：连续点击过快 -2（吓到了）
- 古泉：回答哲学问题错误 -1（但会微笑着说「没关系」）
- 阿虚：永远不会降低（吐槽役的包容）

## 视觉表现

### 好感度条
- 竖向水滴形进度条，位于角色卡片右侧
- 颜色随等级变化：灰→蓝→紫→粉→金→彩虹渐变
- Lv.5以上时，进度条带流光动画

### 互动反馈
- 点击时弹出「+1」「+2」「+3」浮动文字，向上飘散
- 暴击（+3）时文字带金色描边和震动效果
- 升级时角色卡片边框爆发对应颜色的光芒，持续3秒

### 个人线入口
- 好感度达到Lv.4后，角色卡片出现「进入个人线」按钮
- 点击进入全屏叙事界面，模拟Galgame的文本对话框
- 对话框底部有「AUTO」「SKIP」「LOG」三个按钮（纯装饰或有限功能）

## 技术实现建议

### 数据结构
```typescript
interface CharacterAffection {
  characterId: string;
  level: number;           // 当前等级 0-5
  points: number;          // 当前好感度数值
  totalInteractions: number;
  lastInteractDate: string;
  streakDays: number;      // 连续互动天数
  unlockedEvents: string[];
}

interface AffectionState {
  characters: CharacterAffection[];
  globalEvents: string[];  // 已触发的全局事件
}
```

### 存储
- `localStorage` 键名：`sos_affection_state`
- 支持导出/导入存档（JSON文件下载）
- 彩蛋：存档文件命名自动为「SOS团情报统合思念体备份.dat」
