# 扩展功能三：春日团长的异世界任务板

## 功能背景

参考《怪物猎人》的集会所任务板、《原神》的冒险家协会委托、《蔚蓝档案》的总力战，SOS团当然也要有自己的任务系统。春日团长每天都会在黑板上贴满各种「让社团活动更精彩」的任务——从寻找不可思议现象到拍摄电影，从打棒球到调查封闭空间。完成任务可以获得SOS团贡献值，用于在「鹤屋商店」兑换各种虚拟奖励。

> 「任务？这不是任务，这是让世界变得更热闹的必要活动！」——凉宫春日

## 核心概念

### 任务类型

**每日委托（4个/日，每日0点刷新）**
| 任务名 | 类型 | 描述 | 奖励 |
|--------|------|------|------|
| 文艺社教室巡查 | 日常 | 确认文艺社教室没有被其他社团占用 | 100pt |
| 给实玖瑠拍照 | 互动 | 在成员页面与实玖瑠互动3次 | 150pt |
| 长门的读书时间 | 互动 | 查看长门的角色详情页面 | 100pt |
| 团长的突发奇想 | 随机 | 随机生成一个离谱任务（见下方「春日脑洞」） | 200pt |

**周常任务（每周一刷新）**
| 任务名 | 描述 | 奖励 |
|--------|------|------|
| 无尽八月挑战 | 连续7天登录签到 | 1000pt + 特殊称号 |
| SOS团全员集合 | 与所有成员互动至少1次 | 800pt |
| 电影拍摄周 | 在留言墙发布3条以上评论 | 600pt |
| 封闭空间调查 | 累计查看角色详情10次 | 500pt |

**传说任务（一次性）**
| 任务名 | 描述 | 奖励 |
|--------|------|------|
| SOS团创立 | 完成首次登录 | 称号「创始成员」 |
| 我对普通人类没有兴趣 | 首次与春日互动 | 500pt |
| 资讯操作 | 首次与长门互动 | 500pt |
| 这是禁止事项 | 首次与实玖瑠互动 | 500pt |
| 机关的试探 | 首次与古泉互动 | 500pt |
| 吐槽能量MAX | 首次与阿虚互动 | 500pt |
| 凉宫春日的消失 | 在冬季（12月）登录满10天 | 称号「消失世界的幸存者」 |
| 凉宫春日的惊愕 | 连续30天完成所有每日委托 | 称号「超监督」+ 5000pt |

### 春日脑洞（随机每日任务）
春日团长的随机任务完全不讲道理，充满原作梗和二次元梗：

| 脑洞任务 | 实际要求 | 梗来源 |
|----------|----------|--------|
| 「去寻找约翰·史密斯！」 | 在页面任意位置点击搜索10次 | 原作：春日在时间旅行中寻找约翰·史密斯 |
| 「给我抓一只星际迷航里的外星人」 | 与长门互动5次 | 长门就是外星人 |
| 「明天的天气必须晴朗」 | 签到时天气API显示晴天...否则重来 | 原作：春日无意识改变天气 |
| 「让世界重复15532次」 | 刷新页面15532次...开玩笑的，刷新3次就行 | 无尽八月 |
| 「给我拍一部科幻电影」 | 在留言墙发布一条带图片的评论 | 原作：电影拍摄 |
| 「找到SOS团的真正含义」 | 在页面中寻找隐藏的SOS彩蛋（点击团徽3次） | 原作：团名含义 |
| 「组织一场圣战」 | 在留言墙与其他用户争论（其实是发任何评论） | 二次元梗：萌战/圣战 |
| 「证明1+1=3」 | 完成一道简单的数学题目 | 团长的不讲理 |
| 「给实玖瑠穿上兔女郎装」 | 点击实玖瑠的换装图标 | 原作：兔女郎事件 |
| 「调查为什么长门的眼镜会消失」 | 查看长门详情页面的《消失》段落 | 《消失》梗 |

### 任务状态流转
```
[未解锁] → [已解锁/可接取] → [进行中] → [已完成/待领取] → [已领取]
```

- **未解锁**：传说任务需要前置条件（如完成其他任务）
- **已解锁**：任务板上的任务卡片发光，带有「NEW」标签
- **进行中**：显示进度条，如「2/5次互动」
- **已完成**：任务卡片变为金色，带有「！」标记，点击领取奖励
- **已领取**：变为灰色归档状态，移入「已完成」标签页

## 页面设计

### 任务板主界面
- 整体视觉参考《怪物猎人》集会所公告板：木质纹理背景 + 钉在板上的任务单
- 顶部三个标签页：「每日委托」「周常挑战」「传说任务」
- 右下角显示当前SOS团贡献值和「前往鹤屋商店」按钮
- 左上角显示「今日已完成 X/4」进度

### 任务卡片
- 尺寸：类似扑克牌的长方形
- 颜色编码：
  - 蓝色 = 日常委托
  - 紫色 = 周常挑战
  - 金色 = 传说任务
  - 春日脑洞 = 红色（警告色，因为很离谱）
- 正面：任务名 + 类型图标 + 简短描述 + 奖励预览
- 背面（点击翻转）：详细描述 + 当前进度 + 放弃按钮（春日：「放弃？你确定？」）

### 完成任务动画
1. 点击「完成任务」
2. 任务卡片从板上「撕下」的动画（CSS transform 模拟）
3. 贡献值数字跳动增长（类似游戏金币增加效果）
4. 如果完成任务触发了传说任务解锁，显示「新任务解锁！」横幅
5. 连续完成4个每日委托后，春日头像弹出：「今天也是充满干劲的一天！」

## 贡献值与鹤屋商店

### 贡献值（SOS Points）
- 货币单位：「SOSpt」或「团费」
- 显示位置：任务板右下角，导航栏右上角
- 数值变动时有浮动动画

## 视觉风格

- 主色调：木质棕 + 纸张米黄 + 任务类型对应颜色
- 任务单：带钉子和卷角效果的CSS卡片
- 进度条：类似HP条的设计，渐变色填充
- NEW标签：红色圆形徽章，带抖动动画
- 完成标记：金色印章盖在任务单上的效果

## 技术实现建议

### 数据结构
```typescript
interface Quest {
  id: string;
  name: string;
  description: string;
  type: 'daily' | 'weekly' | 'legendary';
  category: string;
  requirements: QuestRequirement[];
  rewards: QuestReward[];
  unlockCondition?: string;
}

interface QuestProgress {
  questId: string;
  status: 'locked' | 'available' | 'in_progress' | 'completed' | 'claimed';
  currentProgress: number;
  targetProgress: number;
  unlockedAt?: string;
  completedAt?: string;
}

interface UserQuestState {
  points: number;
  dailyQuests: QuestProgress[];
  weeklyQuests: QuestProgress[];
  legendaryQuests: QuestProgress[];
  lastDailyReset: string;
  lastWeeklyReset: string;
  inventory: string[];
}
```

### 任务刷新逻辑
- 每日0点（本地时间）刷新4个随机每日委托
- 每周一0点刷新周常任务
- 传说任务一旦解锁永不消失
- 使用 `localStorage` 存储任务状态，键名：`sos_quest_state`
