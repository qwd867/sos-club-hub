# 扩展功能一：SOS团每日签到系统

## 功能背景

参考《原神》《明日方舟》《碧蓝航线》等二次元手游的每日签到系统，SOS团也需要一个打卡机制来保持团员的活跃度。毕竟春日团长可不允许有人摸鱼——「不来参加活动的人，现实可能会被改写哦！」

## 核心概念

### 签到日历
- 以月历形式展示当月签到状态
- 已签到日期显示角色小头像印章（随机五位主角之一）
- 未签到日期显示灰色问号
- 今日可签到日期高亮闪烁，吸引点击

### 连续签到奖励（Streak系统）
| 连续天数 | 称号 | 奖励描述 |
|----------|------|----------|
| 1-2天 | 新人团员 | 获得「普通人类」徽章 |
| 3-6天 | 见习部员 | 获得「鹤屋家的茶」x1 |
| 7天 | 文艺社幽灵 | 解锁长门有希专属问候语 |
| 14天 | 时间旅者 | 解锁朝比奈实玖瑠女仆装壁纸 |
| 30天 | 超能力者 | 解锁古泉一树的哲学小课堂 |
| 15532天 | 无尽八月幸存者 | 致敬原作梗，显示特殊彩蛋 |

### 特殊日期彩蛋
- **12月18日**（《消失》上映日）：签到显示「世界被改写了...但你还活着，太好了」
- **8月17日-31日**（无尽八月期间）：连续签到界面显示循环计数器
- **团长生日**：签到印章强制为春日头像，且带有金色闪光特效
- **2月29日**：显示「这一天不存在于SOS团的记录中...」（闰年梗）

## 页面交互

### 签到弹窗动画
1. 点击「今日签到」按钮
2. 按钮变为加载态，显示春日团长的经典台词：「我对普通的人类没有兴趣！」
3. 弹出全屏特效：屏幕中央出现旋转的SOS团徽章，金色粒子爆发（confetti）
4. 显示签到结果卡片：随机角色印章 + 获得的奖励 + 连续天数
5. 底部显示「分享至鹤屋商店」（装饰性按钮）

### 月历交互
- Hover日期：显示该日期的签到角色名
- 点击已签到日期：回放当日签到动画
- 左滑/右滑切换月份（移动端）
- 月份切换时带有翻页动画效果

## 视觉风格

- 月历卡片：玻璃态圆角矩形，背景 `rgba(255,255,255,0.7)` + `backdrop-filter: blur(16px)`
- 签到按钮：春日黄渐变，带脉冲动画
- 已签到印章：轻微3D倾斜效果，Hover时弹出角色Q版头像
- 连续天数火焰 streak：底部进度条带火焰粒子特效
- 字体：日期数字使用等宽字体，模拟游戏UI风格

## 技术实现建议

### 数据结构
```typescript
interface CheckInRecord {
  date: string;          // YYYY-MM-DD
  character: string;     // 随机角色ID
  streak: number;        // 当日连续天数
  reward?: string;       // 获得的奖励ID
}

interface UserCheckInState {
  currentStreak: number;
  longestStreak: number;
  totalCheckIns: number;
  lastCheckInDate: string;
  records: CheckInRecord[];
}
```

### 存储方案
- 使用 `localStorage` 存储签到数据（纯前端演示）
- 键名：`sos_checkin_state`
- 支持手动「重置数据」按钮（调试/测试用）

### 防作弊
- 检查本地时间是否被篡改（对比上次签到日期）
- 如果检测到时间回退，显示「长门有希：检测到时间悖论」提示
